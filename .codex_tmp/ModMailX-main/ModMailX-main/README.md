## рџ› пёЏ Installation

### Prerequisites
- [Node.js](https://nodejs.org/) v20.0.0 or higher.
- [MongoDB](https://www.mongodb.com/) account for database storage.
- Canvas dependencies (for Linux hosting, ensure `cairo` and `pango` are installed).

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/aryanshrai03/ModMailX.git
   cd ModMailX
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure `config.js`:
   - Add your **Bot Token**.
   - Add your **MongoDB URI**.
   - Set your **Guild ID** and **Staff Role**.
4. Start the bot:
   ```bash
   node foil.js or node .
   ```
